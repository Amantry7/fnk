# ФНК — сайт финансово-консалтинговой компании

Лендинг на **Django (MVT)** + HTML / CSS / JS. Тёмная премиум-тема с золотыми
акцентами под логотип «ФНК». Заявки из формы сохраняются в базу и доступны в
админ-панели.

## Стек
- Python 3.14, Django 5.2 (LTS; 5.2+ обязательна для Python 3.14)
- SQLite (по умолчанию)
- Pillow (обработка логотипа), python-dotenv (настройки через `.env`)
- Чистый HTML/CSS/JS без сборщиков

## Структура
```
fnk/
├── config/            # настройки проекта (settings, urls)
├── main/              # приложение лендинга
│   ├── models.py      # ConsultRequest — заявки
│   ├── forms.py       # форма заявки + honeypot от ботов
│   ├── views.py       # index + submit (AJAX/обычная отправка)
│   ├── admin.py       # админка заявок
│   ├── templates/main/  # base, _header, _footer, index
│   └── static/        # css, js, img (логотип)
├── .env               # переменные окружения
├── requirements.txt
└── manage.py
```

## Запуск
```bash
# 1. активировать окружение (Windows)
venv\Scripts\activate

# 2. (если ставите заново) зависимости
pip install -r requirements.txt

# 3. миграции
python manage.py migrate

# 4. суперпользователь для админки (если ещё не создан)
python manage.py createsuperuser

# 5. запустить сервер
python manage.py runserver
```
Сайт: http://127.0.0.1:8000/
Админка: http://127.0.0.1:8000/admin/

Тестовый администратор уже создан: **admin / admin12345**
(смените пароль перед публикацией).

## Весь контент редактируется через админку
Тексты и блоки сайта хранятся в базе и меняются в админке без правки кода:

- **Настройки сайта** — все одиночные тексты: Hero, заголовки секций,
  «О компании», контакты (телефон, e-mail, адрес, график), подвал, бренд.
  Это одна запись (singleton) — открывается и редактируется.
- **Меню (навигация)** — пункты шапки/подвала.
- **Соцсети** — ссылки в подвале.
- **Hero · статистика**, **О компании · преимущества**, **Факты**,
  **Боли клиентов**, **Услуги**, **Шаги работы**, **Кейсы**, **Тарифы**,
  **Отзывы**, **FAQ** — списки; у каждого есть «Порядок» (сортировка) и
  «Показывать» (скрыть/показать блок без удаления).
- **Заявки** — обращения с формы: фильтр по статусу/формату/дате, поиск,
  отметка «обработана».

Начальный контент (заглушки) уже залит миграцией — заходите в админку и
заменяйте на реальный.

### Тарифы
Поле «Состав» — каждый пункт списка с новой строки. «Выделенный» — золотая
карточка с бейджем. «Ключ формата» подставляется в форму заявки при клике
на кнопку тарифа.

## Логотип
`main/static/img/` — `logo-sm.png` (шапка), `logo-full.jpg` (Hero);
оригинал `Фнк_gold.png`. Замена логотипа — только через файлы.

## Перед публикацией (продакшен)
В `.env`:
- сгенерируйте новый `SECRET_KEY`
- `DEBUG=False`
- укажите домен в `ALLOWED_HOSTS`
- выполните `python manage.py collectstatic`
- настройте отправку заявок на почту/Telegram при необходимости
